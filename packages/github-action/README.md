# CodeTitan GitHub Action Package

The public action entrypoint for consumers is the repository root:

- `uses: Noa-Lia/codetitan-action@v1`

This package directory is kept so the bundled runtime can still locate `packages/core` and pack the internal core engine on clean GitHub-hosted runners.

For usage, supported scope, and public proof links, see the root [README](../../README.md).
